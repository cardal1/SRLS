
To run the SRLS algorithm under windowsOS as follows:
.\SRLS.exe seed instance_file result_file
/*example*/    .\SRLS.exe 1 .\test_instances\QWH-50-70-1.txt .\res.txt

To run the validator under windowsOS as follows:
.\validator.exe instance_file result_file
/*example*/    .\validator.exe .\test_instances\QWH-50-70-1.txt .\res.txt
